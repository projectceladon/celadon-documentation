The ACRN project defines a device `Hypervisor <https://github.com/projectacrn/acrn-hypervisor>`_ that provides a virtual machine manager for securely running multiple operating systems on a consolidated system, and a `Device Model <https://github.com/projectacrn/acrn-hypervisor/tree/master/devicemodel>`_ that provides a reference framework for virtual device emulation.
