Intel NUC Support

The Intel® NUC is a powerful mini PC with entertainment, gaming, and productivity features, including a customizable board that can accept the memory, storage, or operating systems that you need.

Intel Movidius Neural Compute Stick

The Intel® Movidius™ Neural Compute Stick (NCS) is a tiny fanless deep learning device that you can use to learn AI programming at the edge. To learn more on how the NCS can be used, check out the NCS developer blog.

Linux Kernel

Kernel.org is the main source for the Linux kernel, which is the base of all Linux operating systems.

Android Open Source Project

Android Open Source Project (AOSP) is the open source Android software stack led by Google.

Project ACRN

ACRN* is a flexible, lightweight reference hypervisor, built with real-time and safety-criticality in mind, optimized to streamline embedded development through an open source platform. The ACRN project defines a device Hypervisor that provides a virtual machine manager for securely running multiple operating systems on a consolidated system, and a Device Model that provides a reference framework for virtual device emulation.
