Android 11 Stable Releases IoT

Android 11 Stable Releases (IoT) is an Internet of Things (IoT) overlay on top of Celadon. The Stable Releases (IoT) release provides users with an extensive set of features on Comet Lake (CML), Tiger Lake (TGL), and Elkhart Lake (EHL) platforms with a single Baseline/Image of Android 11

  • Feature Delta Offered by Stable Releases (IoT)
    • Real-time clock (RTC) and alarm virtualization
    • Secure data erase
